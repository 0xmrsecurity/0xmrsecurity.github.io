# System & Identity
> Identify the Domain Name and Domain IP Addresses.
>

```bash
[Domain Name]
nxc smb $target
nmap -sCV -T4 -vv -Pn -A $target

[DC IP]
host $Domain
nmcli dev show <interface>

[IPv6]
python3 IOXIDResolver.py -t  192.168.xx.xx
```
