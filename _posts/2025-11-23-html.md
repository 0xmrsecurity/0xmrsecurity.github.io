### iframe
> Embedding External Content: The src attribute specifies the file of the document to embed.

1. Try to grab the id_rsa key `home/<username>/.ssh/id_rsa`
2. Try to find the creds locations of the application.
3. Important file locations:- 
`/proc/self/cmdline`
`/proc/self/environ` 
`/etc/apache2/sites-enabled/000-default.conf`
`/etc/nginx/sites-enabled/default`
`/var/log/apache2/access.log`
`/var/log/auth.log`
